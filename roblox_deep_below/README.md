# Deep Below — Roblox Studio Scaffold

Fresh Roblox project scaffold for an underwater survival + exploration game.

## Defaults Applied
- Full systems scaffold (light content)
- Multiplayer-ready
- Monetization hooks stubbed
- Base building + vehicle spawning stubs
- Inventory, crafting, shop, quests, and progression UI
- Base storage UI (deposit/withdraw)

## How To Use
1. Create a new Baseplate place in Roblox Studio.
2. Follow `docs/InstanceManifest.md` to set up folders/instances.
3. Paste scripts from `src/` into matching instances.
4. Press Play. A basic loop (oxygen, light, loot, depth zones) is wired.

## UI Keys
- `I`: Inventory
- `K`: Crafting
- `B`: Shop
- `V`: Vehicles
- `H`: Base building
- `Q`: Quests
- `P`: Progression
- `R`: Rotate base build
- `F`: Basic boss attack (debug)
